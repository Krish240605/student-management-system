# Student Management System

A simple Student Management System developed using MySQL to manage student records, courses, attendance, and grades. The project demonstrates database design, normalization, CRUD operations, SQL queries, and report generation.

## Features

* Manage student information
* Course and enrollment management
* Attendance tracking
* Grade management
* SQL-based reports and analytics

## Technologies Used

* MySQL
* SQL
* ER Diagram Design

## Learning Outcomes

* Database schema design
* Data normalization
* Foreign key relationships
* CRUD operations
* SQL joins and aggregate queries
* Report generation
